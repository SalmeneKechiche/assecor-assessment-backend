# Application

Das Application-Layer implementiert die Geschäftslogik und orchestriert den Datenfluss zwischen API und Domain.

## Inhalt

- **Services**: Geschäftslogik-Implementierungen (PersonService)
- **DTOs**: Datenübertragungsobjekte für API-Kommunikation

## Funktionen

- Person-Verwaltung (CRUD-Operationen)
