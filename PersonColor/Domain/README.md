# Domain

Das Domain-Layer enthält die Kerngeschäftslogik und Entitäten der PersonColor-Anwendung.

## Inhalt

- **Entities**: Geschäftsobjekte (Person)
- **Interfaces**: Abstrakte Definitionen für Repository-Verträge
- **ColorMapping**: Statische Klasse für Farbzuordnungen (1-7 zu Farbnamen)

## Abhängigkeiten

Keine externen Abhängigkeiten - reine Geschäftslogik.
